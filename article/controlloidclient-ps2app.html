ControlloidClient-一款开源移动模拟PS2的APP

Created At：2025-06-01 11:20 Updated At：2025-06-25 21:31
Keywords: 游戏 APP

一款开源移动应用程序，适用于 PC 游戏体验，支持 Linux 和 Windows 系统。可通过 WiFi、蓝牙、USB 等多种连接方式将手机与电脑无缝连接，甚至支持通过互联网远程控制。其核心功能包括模拟 PS2 控制器的所有按钮（除 L3 和 R3 外），支持多客户端连接，以及可自定义的模拟摇杆死区设置等。

Controlloid - Client 可以通过以下方式下载：

  • F-Droid：可以访问F-Droid 官方网站，在页面中找到 Controlloid - Client 进行下载。推荐安装 F - Droid 客户端，通过客户端下载安装 Controlloid - Client，这样能接收更新通知且更安全。
  • GitHub6：访问Controlloid - Client 的 GitHub 页面，在页面中查找相关的下载链接或说明进行下载。4
