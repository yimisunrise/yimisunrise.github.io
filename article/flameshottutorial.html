功能特点

  • 开源免费：跨平台支持（Linux/Windows/macOS）
  • 轻量高效：快速启动，低内存占用
  • 丰富标注工具：箭头、矩形、圆形、文字、模糊、高亮等
  • 快捷键支持：自定义截图快捷键
  • 即时分享：支持保存到本地或直接上传图床
  • 贴图功能：将截图悬浮在桌面随时查看

安装方法

Linux 系统

# Ubuntu/Debian
sudo apt install flameshot

# Fedora
sudo dnf install flameshot

# Arch
sudo pacman -S flameshot

macOS

brew install --cask flameshot

Windows

  • 从 GitHub Releases 下载安装包

基础功能介绍

（示意图：标注工具栏展示）

工具 快捷键 说明
矩形选区 R 截取指定区域
箭头 A 添加指示箭头
画笔 P 自由绘制线条
文本 T 插入文字标注
模糊 B 打码敏感信息
撤销 Ctrl+Z 撤销上一步操作
保存 Ctrl+S 保存截图到本地

使用步骤

步骤1：启动程序

  • 命令行启动：终端输入 flameshot gui
  • 快捷键启动：默认 PrintScreen 键（可自定义）

步骤2：选择截图区域

  1. 拖动鼠标选择截图范围
  2. 按 Enter 确认选区

步骤3：使用标注工具

  • 点击工具栏图标或使用快捷键添加标注
  • 示例：用箭头+文字说明重点内容

步骤4：保存或分享

  • 保存本地：点击磁盘图标或按 Ctrl+S
  • 复制到剪贴板：点击剪贴板图标或按 Ctrl+C
  • 上传图床：点击云上传图标（需配置API）

常见问题

Q1：截图后无法保存？

  • 检查文件保存路径权限
  • 尝试更换保存路径：设置 -> 保存 -> 修改默认路径

Q2：快捷键冲突？

  • 打开设置页面：flameshot config
  • 修改快捷键：General -> 快捷键

Q3：如何模糊敏感信息？

  1. 点击工具栏「模糊」工具（或按 B）
  2. 涂抹需要隐藏的区域
  3. 支持调整模糊强度：右键选择模糊程度

结语

Flameshot 是一款兼具效率与功能的截图工具，通过快捷键和丰富的标注工具，能显著提升工作/学习场景中的信息传达效率。立即体验，让截图编辑变得更专业！
