以下是一篇简单通用的Joplin开源笔记软件图文教程：

Joplin 入门图文教程

一、什么是Joplin？

Joplin 是一款免费开源的笔记应用，支持： ✅ Markdown格式 ✅ 多平台同步（Windows/macOS/Linux/iOS/Android） ✅ 端到端加密 ✅ 笔记本层级管理 ✅ 离线使用

二、安装与设置

  1. 官网下载：访问 https://joplinapp.org
  2. 选择对应系统版本安装

三、基础操作

1. 创建笔记本

  • 点击左侧栏 ➕ 按钮
  • 输入笔记本名称（例如：“工作笔记”）
  • 支持多级嵌套（右键创建子笔记本）

2. 新建笔记

  1. 选中目标笔记本
  2. 点击顶部工具栏 “新建笔记” 按钮
  3. 输入标题和内容：
   # 这是标题
   ## 二级标题
   - 列表项
   **加粗文本**

3. Markdown快捷操作

快捷键 功能
Ctrl+B 加粗文本
Ctrl+I 斜体文本
Ctrl+K 插入链接
Ctrl+Shift+I 插入图片

四、同步设置（以WebDAV为例）

  1. 点击菜单栏：工具 > 同步
  2. 选择同步目标：
    • Dropbox
    • OneDrive
    • WebDAV
    • 本地文件夹
  3. 输入账号信息完成授权
  4. 点击立即同步按钮 🔄

五、高级功能

1. 添加标签

  • 在笔记编辑界面右侧栏
  • 点击 “添加标签” 创建分类
  • 支持通过标签筛选笔记

2. 待办事项

- [ ] 未完成任务
- [x] 已完成任务

3. 搜索功能

  • 使用 Ctrl+F 快速搜索
  • 支持语法：tag:重要 notebook:工作

六、移动端使用

  1. 在应用商店搜索 Joplin
  2. 安装后配置同步设置（需与桌面端相同）
  3. 支持相机扫描文档 📷

七、数据导出

  • 支持导出为：
    • Markdown
    • PDF
    • HTML
    • JEX（Joplin专用格式）

小贴士： 🔹 使用 Ctrl+/ 切换Markdown源码视图 🔹 定期使用 “检查同步配置” 功能 🔹 重要笔记建议启用加密

官方资源：

  • 用户手册：https://joplinapp.org/help/
  • 论坛支持：https://discourse.joplinapp.org

通过这个教程，你可以快速开始使用Joplin进行知识管理。随着使用深入，可以逐步探索更多高级功能！
